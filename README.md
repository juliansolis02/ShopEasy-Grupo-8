# ShopEasy-Grupo-8
ShopEasy es una plataforma web de comercio electrónico con carrito de compras, orientada a pequeñas y medianas empresas que necesitan gestionar productos, pedidos, inventario y usuarios desde una aplicación web.

## Tecnologías previstas

- Java
- Spring Boot
- Thymeleaf
- Bootstrap
- Hibernate / JPA
- MySQL
- GitHub

## Funcionalidades principales

- Registro e inicio de sesión de usuarios.
- Gestión de roles: administrador, empleado y cliente.
- Administración de productos y categorías.
- Carrito de compras.
- Gestión de pedidos.
- Control de inventario.
- Reportes básicos de ventas.
- Soporte multiidioma.

## Estrategia de ramas

- `main`: rama principal donde se mantendrá la versión estable del proyecto.
- `develop`: rama de integración para unir los avances del equipo antes de pasarlos a producción.
- `feature/nombre-funcionalidad`: ramas utilizadas para desarrollar funcionalidades específicas.
- `hotfix/nombre-correccion`: ramas utilizadas para corregir errores importantes.
