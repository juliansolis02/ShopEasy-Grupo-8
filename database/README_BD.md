1. Crear una base de datos llamada:

   shopeasy_db

2. Configurar las credenciales de MySQL en el archivo:

   src/main/resources/application.properties

   Ajustar según la configuración local del usuario.

3. Ejecutar el proyecto para que Hibernate cree las tablas.

4. Ejecutar el script:

   database/shopeasy_db.sql

5. Reiniciar la aplicación.
